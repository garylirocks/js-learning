RequireJS Demo - Number Guessing

Loading process:

  1. 'require.js' is loaded, it finds the entry JS file is 'main.js';
  2. 'main.js' is loaded, which requires 'NumberGuessing.js';
  3. 'NumberGuessing.js' is loaded, which requires 'RandomNumber.js';
  4. 'RandomNumber.js' is loaded, everything is loaded now;
  5. function in 'main.js' starts;
